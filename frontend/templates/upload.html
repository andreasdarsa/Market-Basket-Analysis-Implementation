CSV File Instructions

Please ensure your CSV file is formatted correctly:

  • The first row should just contain column headers.
  • Each subsequent row should represent a transaction.
  • Items in each transaction should be separated by commas.
  • The first column should be TransactionID
  • The second column should be Items (items separated by commas)
  • Example format:
  • TransactionID,Items
  • 1,Milk,Bread,Eggs
  • 2,Bread,Butter
  • 3,Milk,Eggs
  • Ensure there are no empty rows or columns.
  • Save the file with a .csv extension.
  • For more details, refer to the About page.
